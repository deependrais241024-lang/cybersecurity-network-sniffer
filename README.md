# Network Sniffer Project

A Python-based network packet sniffer tool that captures, filters, and analyzes live network traffic using Scapy.  
This project is developed for cybersecurity internship learning and demonstrates basic network monitoring concepts.

---

## Features
- Captures live network packets
- Displays IP, TCP, and UDP details
- Filters network traffic
- Saves captured packets into log file

---

## Files Included
- sniffer.py → Main Python script
- log.txt → Captured packet logs
- .gitignore → Ignored files configuration
- Screenshot.png → Output proof (optional)

---

## Tools Used
- Python
- Scapy

---

## Purpose
This project is created for cybersecurity internship training to understand network traffic analysis and packet inspection.
